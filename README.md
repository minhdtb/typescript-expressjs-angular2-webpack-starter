# Starter minimalist  web application using TypeScript, ExpressJS, Angular 2 and Webpack

1. Automates the task with Webpack
2. Both client and server are using TypeScript
3. Uses `ejs` as the templating engine

**Setup**
---
```
npm install
```

**Run Builds**
---
```
npm run build
```

**Run Application**
---
```
npm start
```
